# iCost Analyzer

iCost Analyzer 是一个用于分析 iCost 导出账单的开源小工具。你可以把 iCost
导出的 `.xlsx` / `.xls` 文件拖进页面，在浏览器里查看支出、收入、分类、
标签、趋势、热力图和交易明细。

> 这是一个第三方非官方项目，与 iCost 官方无关联。账单文件只在当前浏览器中
> 解析，不会上传到服务器。

## 适合谁使用

- 想快速回顾一段时间内自己把钱花在哪里的人。
- 已经在 iCost 记账，希望用更大的屏幕查看趋势、分类和明细的人。
- 需要临时分析多币种消费、旅行账单、年度账单或分类支出的人。
- 关心隐私，希望账单文件留在自己电脑上的人。

## 你可以看到什么

- 总支出、总收入、净结余、月均支出、日均支出和最大单笔支出。
- 月度趋势、分类占比、支出排行、币种分布、星期分布、标签排行和每日热力图。
- 分类汇总、标签汇总和可排序、可分页的交易明细。
- 按时间、年份、日期区间、交易类型、币种、分类、标签和关键词筛选。
- 点击图表中的月份、分类或标签后，联动筛选其他数据。
- 浅色、深色和跟随系统的主题模式。

## 如何使用

1. 在 iCost 中导出 Excel 格式的账单。
2. 打开 iCost Analyzer。
3. 把 `.xlsx` / `.xls` 文件拖到页面中，或点击上传区域选择文件。
4. 等待浏览器完成解析。
5. 查看指标、图表和明细，按需要调整筛选条件。
6. 如果账单里有外币消费，可以在汇率区域手动调整折算汇率。

焦点不在输入框时，可以按 `d` 快速切换深色/浅色主题。

## 账单文件要求

应用会读取 Excel 的第一个工作表。至少需要包含可识别的日期和金额列；其他
字段缺失时会使用默认值。

| 含义     | 是否必需 | 支持的列名                 | 缺省值         |
| -------- | -------- | -------------------------- | -------------- |
| 日期     | 是       | `日期`、`日期时间`、`时间` | 无效时跳过该行 |
| 金额     | 是       | `金额`、`原始金额`         | 无效时跳过该行 |
| 币种     | 否       | `货币`、`币种`、`Currency` | `CNY`          |
| 类型     | 否       | `类型`、`交易类型`         | 按金额正负推断 |
| 一级分类 | 否       | `一级分类`、`分类`         | `未分类`       |
| 二级分类 | 否       | `二级分类`、`子分类`       | `未分类`       |
| 标签     | 否       | `标签`、`Tags`             | 空标签         |
| 备注     | 否       | `备注`、`说明`             | 空文本         |
| 地点     | 否       | `位置`、`地点`             | 空文本         |

解析时会遵循这些规则：

- 日期或金额无法识别的行会被跳过。
- 没有交易类型时，负数金额会被视为 `支出`，非负数金额会被视为 `收入`。
- 没有币种时使用 `CNY`，已有币种会自动转成大写。
- 没有分类或子分类时显示为 `未分类`。
- 标签支持中文/英文逗号、顿号、分号、竖线和空格分隔，也支持 `#标签` 写法。

## 汇率说明

- `CNY` 是基准币种，汇率固定为 `1`。
- 默认汇率是应用内置的静态参考值，不会联网获取实时汇率。
- 你可以在页面中手动修改非人民币币种的汇率。
- 空值、非法值或非正数汇率不会覆盖当前有效汇率。

## 隐私说明

- 账单文件在浏览器本地解析，不会上传到服务器。
- 项目目前没有后端服务，也没有远程解析或遥测逻辑。
- 不建议把真实账单文件、截图或包含个人信息的示例提交到公开仓库。

## 本地运行

如果你想在自己的电脑上运行这个开源项目，需要先安装 Node.js LTS 和 `pnpm`：

```bash
pnpm install
pnpm dev
```

启动后，打开终端中显示的本地地址即可使用。
