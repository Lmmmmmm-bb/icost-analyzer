# iCost Analyzer

iCost Analyzer 是一个开源、第三方、非官方的 iCost 账单分析工具。你可以把
iCost 导出的 `.xlsx` 或 `.xls` 账单导入页面，在自己的浏览器里查看收支概览、
消费趋势、分类占比、标签排行、账户分布、币种折算和交易明细。

本项目不隶属于 iCost 官方，也不代表 iCost 官方提供服务。账单文件只在当前
浏览器页面中解析，不会上传到服务器；默认汇率是应用内置的静态参考值，不是
实时市场汇率。

## 你可以用它做什么

- 快速查看总支出、总收入、净结余、月均支出、日均支出、笔均支出、最大单笔
  支出和退款/报销金额。
- 通过月度趋势、分类占比、分类排行、账户支出分布、账户支出排行、币种分布、
  星期分布、标签排行、每日支出热力图和每日收支截面理解账单变化。
- 按年份、日期区间、交易类型、币种、账户、分类、标签和关键词筛选账单。
- 点击图表里的月份、分类、账户或标签，联动查看对应交易明细。
- 手动调整外币汇率，把不同币种的金额统一折算成人民币查看。
- 在浅色、深色和跟随系统主题之间切换。

## 使用方法

1. 在 iCost 中导出 Excel 格式账单。
2. 打开 iCost Analyzer 页面。
3. 将 `.xlsx` 或 `.xls` 文件拖到页面任意位置，或点击上传区域选择文件。
4. 等待浏览器完成解析；账单越大，解析时间可能越长。
5. 查看指标、图表和明细，并按需要调整筛选条件。
6. 如果账单包含外币，在“汇率”区域检查或修改折算汇率。
7. 需要分析另一份账单时，直接拖入新文件，或点击“更换文件”。

导入后的账单数据只保存在当前页面中。刷新页面、关闭页面或更换文件后，需要
重新导入账单。

## 隐私说明

- 账单文件在浏览器本地解析，不会上传到服务器。
- 项目目前没有后端服务、远程解析、账号系统或遥测逻辑。
- 除主题偏好外，应用不会把账单数据写入浏览器存储。
- 请不要把真实账单文件、私人截图或包含个人信息的示例提交到公开仓库。
